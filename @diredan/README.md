## Installation

1) `git clone https://github.com/DireDan/DiscordJS_Bot.git`
2) `cd DiscordJS_Bot`
3) `npm install`
4) open up config.json
5) replace INSERT_TOKEN with your bots token
6) replace INSERT_YOUR_USER_ID with your user id
7) `node index.js`
8) success!
